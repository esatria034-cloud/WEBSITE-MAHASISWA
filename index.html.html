📖 Guidelines & Konsep OOP (Object-Oriented Programming)
Aplikasi SIMADIK ini dibangun mengikuti 4 pilar utama OOP — Enkapsulasi, Pewarisan (Inheritance), Polimorfisme, dan Abstraksi — dipadukan dengan konvensi penamaan yang konsisten, validasi input berbasis regex, dan penanganan error melalui try-catch.
🏷 Penamaan Variabel
// ✅ camelCase untuk variabel & fungsi let studentList = []; const MAX_IPK = 4.0; // SCREAMING_SNAKE konstan function validateInput(data) { ... } // ❌ Hindari nama samar let x, a, tmp2;
🧱 OOP: Enkapsulasi
class Mahasiswa { // Private fields (#) — tidak bisa diakses langsung dari luar class #nim; #nama; #ipk; constructor(nim, nama, ipk) { this.#nim = nim; } // Getter — akses terkontrol untuk membaca data get nim() { return this.#nim; } // Setter — validasi sebelum mengubah data set nama(v) { if (v.trim().length >= 3) this.#nama = v; } }
🔗 OOP: Pewarisan (Inheritance)
class MahasiswaBerprestasi extends Mahasiswa { // Field tambahan khusus subclass #beasiswa; constructor(nim, nama, email, prodi, angkatan, ipk, beasiswa) { // super() memanggil constructor parent class super(nim, nama, email, prodi, angkatan, ipk); this.#beasiswa = beasiswa; } }
🎭 OOP: Polimorfisme
// Method getInfo() di parent class class Mahasiswa { getInfo() { return `[${this.#nim}] ${this.#nama} — IPK ${this.#ipk.toFixed(2)}`; } } // Subclass meng-override method yang sama // dengan perilaku berbeda (polimorfisme) class MahasiswaBerprestasi extends Mahasiswa { getInfo() { return super.getInfo() + ` 🏆 (${this.#beasiswa})`; } } // Saat dipanggil, objek otomatis menjalankan // versi getInfo() miliknya sendiri students.forEach(s => console.log(s.getInfo()));
🧩 OOP: Abstraksi & Static Method
class Mahasiswa { // Method statis: helper bawaan class itu sendiri, // tidak butuh instance untuk dipanggil static fromJSON(o) { return new Mahasiswa(o.nim, o.nama, o.email, o.prodi, o.angkatan, o.ipk); } } // Pengguna cukup tahu "apa" yang dilakukan getInfo()/toJSON(), // tanpa perlu tahu detail implementasinya — inilah abstraksi. const data = Mahasiswa.fromJSON(jsonObj);
🛡 Regex & Try-Catch
